# kirakira-eye
星空のように瞬く小さな球体が画面に浮遊するシンプルな3Dアニメーションです。

## デモ
https://js.sabae.cc/kirakira-eye.html

## 機能
- 画面上に8個(設定可能)の白い球体を表示
- それぞれの球体が独自のパスを描きながら、ゆっくりと移動する

## 必要環境
- 任意のWebブラウザ

## 使い方
ブラウザでHTMLファイルを開くと、すぐに動作を確認できます。  
URL末尾にパラメータ(球体の個数,各球体の大きさ)を指定することで、表示を変更できます。  
例: https://js.sabae.cc/kirakira-eye.html#12,0.3
