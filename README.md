# kirakira-eye

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

A simple WebGL scene with floating spheres.

## Demo
View the demo at the [GitHub page](https://js.sabae.cc/kirakira-eye/).

## Features
- Animated floating spheres
- Customizable number and size of spheres

## Usage
To run the project, simply open the `index.html` file in a web browser.

You can customize the number of spheres (N) and their size (SIZE) by modifying the URL hash. For example:

`index.html#12,0.7`

This will create 12 spheres with a size of 0.7.
